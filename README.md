# kotoku-in
ブロックチェーンの初学者向けにわたしが実際に使ったコードを公開します。
名前の高徳院は、鎌倉の大仏殿に由来しています。

## 前提知識
- npmを使ったパッケージインストールができる
- TCP/IPにおけるポート番号やDNSの意味がわかる

## 動作環境
- MacOS 11.3
  - Intel Core i7 (M1 Macは動作未確認)
- Node v15.12.0
